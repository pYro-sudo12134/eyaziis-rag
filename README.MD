# RAG Agent — система семантико-синтаксического анализа текста

RAG-агент с веб-интерфейсом для анализа текста на русском языке. Поддерживает загрузку документов, синтаксический разбор, выделение семантических ролей и диалог на основе базы знаний.

## Технологии

- **Backend**: Python 3.11.9, boto3, Flask, opensearch-py
- **Векторная БД**: OpenSearch
- **LLM**: Ollama (qwen2.5:7b)
- **Эмбеддинги**: nomic-embed-text
- **Хранилище**: LocalStack (S3)
- **Контейнеризация**: Docker Compose
- **GPU**: NVIDIA CUDA
