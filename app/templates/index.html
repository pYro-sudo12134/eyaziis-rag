О системе

RAG Agent - интеллектуальная система анализа текста c использованием Retrieval-Augmented Generation (RAG). Система сочетает семантический поиск по загруженным документам с генерацией ответов на основе LLM.

Возможности

  • Загрузка документов - поддерживаются TXT, PDF, DOCX, DOC, HTML, RTF форматы
  • Синтаксический анализ - построение дерева зависимостей предложения
  • Семантический анализ - выделение семантических ролей (агент, пациенс, инструмент, локация)
  • Семантические отношения - выявление гиперонимов, гипонимов, синонимов, причинно-следственных связей
  • Сохранение результатов - экспорт анализа в S3 хранилище
  • Предметные области - настройка контекста для более точных ответов
  • Редактирование сообщений - изменение текста любых сообщений
  • Удаление сообщений - удаление отдельных сообщений из истории
  • Перегенерация ответов - пересоздание ответов ассистента
  • Редактирование деревьев - ручное редактирование синтаксических деревьев

Команды

/help Показать справку
/syntax [текст] Построить синтаксическое дерево для текста
/semantic [текст] Выполнить семантический анализ
/save Сохранить текущий анализ
/stats Показать статистику системы
/clear Очистить историю диалога

Часто задаваемые вопросы

Как редактировать сообщение?
Наведите курсор на сообщение и нажмите кнопку "Редактировать". После редактирования сообщение автоматически сохранится в истории.
Как перегенерировать ответ?
Наведите курсор на ответ ассистента и нажмите кнопку "Перегенерировать". Система создаст новый ответ на основе того же вопроса.
Как редактировать синтаксическое дерево?
Наведите курсор на ответ ассистента и нажмите кнопку "Редактировать дерево". Откроется редактор JSON, где можно изменить структуру дерева.
Какие форматы документов поддерживаются?
Поддерживаются TXT, PDF, DOCX, DOC, HTML, RTF. Максимальный размер файла - 50 МБ.
Как сохранить результаты анализа?
Нажмите кнопку "Сохранить" в блоке синтаксического или семантического анализа. Результаты сохраняются в S3 хранилище и доступны в разделе "Сохраненные результаты".
Что такое семантические роли?
Семантические роли описывают функции слов в предложении: Агент (кто выполняет действие), Пациенс (на кого направлено действие), Инструмент (чем выполняется), Локация (где), Время (когда) и другие.

API Endpoints

  • POST /api/chat - основной чат c RAG
  • POST /api/upload - загрузка документа
  • POST /api/semantic_analysis - полный семантико-синтаксический анализ
  • GET /api/stats - статистика системы
  • POST /api/save_result - сохранение результатов
  • POST /api/update_message - обновление сообщения
  • POST /api/delete_message - удаление сообщения
  • POST /api/regenerate_response - перегенерация ответа
  • POST /api/update_syntax_tree - обновление синтаксического дерева
